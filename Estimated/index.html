STORAGE

選擇資料儲存方式

資料只會保存在你的裝置，不會上傳。大多數玩家使用瀏覽器儲存即可；想自行管理存檔檔案時，可選擇本機 JSON。

本機 JSON 檔案儲存需要 Chrome / Edge 桌面版，並透過 localhost 或 HTTPS 開啟；不支援時仍可使用瀏覽器儲存與匯出 / 匯入備份。
